# My AI Toolbox

一个面向非技术用户的中文 AI 工具导航网站，帮助用户发现、筛选、收藏并访问常用 AI 工具。

## 在线访问

https://xxwhh-lab.github.io/my-ai-toolbox/

## 当前功能

- 展示国内外主流 AI 工具
- 支持按工具名称和简介关键词搜索
- 支持按分类筛选
- 支持搜索与分类组合筛选
- 支持星星收藏和取消收藏
- 支持“仅看收藏”
- 使用 `localStorage` 保存收藏状态
- 每个工具均可跳转到官网
- 官网在新标签页打开
- 支持无结果和无收藏时的空状态提示
- 适配电脑和手机屏幕
